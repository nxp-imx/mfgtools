# libuuu

A python wrapper for `libuuu`.

Supported Python versions: ?

## Installation

Install from sources:

```bash
    $ git clone <url>
    $ cd wrapper
    # Create and activate virtual environment:
    $ python -m venv venv
    # Activate virtual env
    $ venv\Scripts\activate
    # on Linux/Mac use `source venv/bin/activate`
    $ pip install -U -e .
```
